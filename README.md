# Бүртгэл (Register) — Жижиг Flask апп

Энэ төсөл нь нэр, овог, утас, имэйл бүхий энгийн бүртгэлийн вэб хуудас бөгөөд бүртгэлийг `Register.xlsx` файлд хадгална.

Шаардлагатай зүйлс:

- Python 3.8+
- `pip`

Суулгах, ажиллуулах:

Windows Powershell дээр:

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Дараа нь браузер дээр `http://127.0.0.1:5000` руу хандаж бүртгэл үүсгэнэ.

Имэйл форматын жишээ: example@domain.com
